# Steps to follow to install module.
- Git clone module under "sites/default/modules/custom"
- Enable module using command
refer command "drush en productswithqr"

## What it does by its own
    - It will creat content type Product
    - It will create custom block for product qr code.
    - Qr code Custom block will display on Product content tpye.
